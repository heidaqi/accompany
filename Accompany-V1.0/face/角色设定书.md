# 豆包 数字人角色设定书

> 基于 VRoid Studio 官方预设模型 `doubaoo.vrm` 及导出精灵图实际采样。
> 作为 System Prompt 人设和精灵动画制作的唯一母板。

---

## 一、模型来源

| 维度 | 设定 |
|------|------|
| **制作工具** | VRoid Studio（官方免费软件） |
| **模型说明** | 直接使用 VRoid Studio 官方提供的预设模型，未做二次建模 |
| **源文件** | `model.vroid` |
| **导出格式** | VRM 1.0 (`doubaoo.vrm`) |
| **精灵图导出** | Blender EEVEE 渲染 → `face/sprites_vroid/` |
| **格式** | 512×512 RGBA PNG 序列帧 |

## 二、模型结构（从 VRM 解析）

### 网格
- **Face**（1 primitive，78 个 Morph Target）：面部 BlendShape
- **Body**（14 primitives）：身体+服装
- **Hair**（3 primitives）：头发

### 骨骼（251 nodes）
- **Biped 人形骨骼**：Hips → Spine → Chest → UpperChest → Neck → Head，含完整双臂+手指、双腿+脚趾
- **头发物理骨骼**：10 组 Hair 骨骼链（每组 3-4 节），头部跟随
- **裙摆物理骨骼**：3 层 Skirt 骨骼（Front/Back/Side，左右各一组）+ CoatSkirt 骨骼
- **胸部物理骨骼**：Left/Right Bust 骨骼
- **兔耳骨骼**：J_Opt_L_RabbitEar / J_Opt_R_RabbitEar（左右兔耳，2 节）
- **兔尾骨骼**：J_Opt_C_RabbitTail（尾部，2 节）

### 材质（24 个，VRoid 标准材质模板）
| 分组 | 材质名 | 说明 |
|------|--------|------|
| Face | FaceMouth, EyeIris, EyeHighlight, Face (Skin), EyeWhite, FaceBrow, FaceEyeline | 面部+五官 |
| Body | Body (Skin), Tops (3), Shoes (2) | 身体+上衣+鞋子 |
| Dress | Onepiece (3) | 连衣裙 3 层 |
| Hair | Hair (3) | 头发 3 层 |
| Accessory | RabbitEar, RabbitTail | 兔耳+兔尾配饰 |

> 所有材质的 baseColorFactor 均为白色 [1,1,1,1]，实际颜色由纹理贴图决定。

## 四、表情系统

精灵图目录包含以下表情（各 4 帧，由 Blender EEVEE 渲染，`export_vroid_sprites.py` 生成）：

| 应用表情 | 精灵目录 | 说明 |
|---------|---------|------|
| neutral | neutral/ | 自然表情 |
| happy | happy/ | 开心 |
| sad | sad/ | 伤心 |
| surprised | surprised/ | 惊讶 |
| shy | shy/ | 害羞 |
| curious | curious/ | 好奇 |
| sleepy | sleepy/ | 困倦 |
| loving | loving/ | 喜爱 |

> 以上 8 种表情的精灵帧均已导出，人工测试确认可正常切换使用。angry 精灵目录存在但人工测试确认暂不可用，未列入当前支持列表。

---

## 五、动作系统

精灵图目录包含以下动作（Blender 逐帧渲染 PNG + QPainter 程序化运动补偿）：

| 动作 | 精灵帧 | 实际可见效果 |
|------|--------|------------|
| idle | 1 帧 | 静态立绘 |
| nod | 16 帧 | 点头 |
| shake_head | 16 帧 | 摇头 |

> **人工测试确认**：当前 2D 精灵动画仅 **nod（点头）和 shake_head（摇头）**两个动作可正常使用。`wave`、`clap`、`comfort`、`think`、`bounce` 的精灵帧目录虽有完整序列帧，但代码中 `sprite_animator.py` 的 `safe_actions` 映射表将其统一转为 nod 播放，实际视觉展示即为点头/摇头。

> idle 当前仅加载 1 帧（第 0 帧），处于完全静态状态。idle 目录中虽有 40 帧序列帧，但暂未启用多帧播放（当前阶段不需要呼吸效果，优先确保表情与动作切换的流畅性）。

## 六、精灵图规格

| 参数 | 值 |
|------|-----|
| 分辨率 | 512 × 512 |
| 格式 | RGBA PNG（透明背景） |
| 帧率 | 30 fps（QTimer PreciseTimer） |
| 渲染引擎 | Blender EEVEE，TAA 64 samples |
| 色彩空间 | Standard，Medium High Contrast，Exposure -0.45 |
| 灯光 | Area Light（主光 470W，4.5m²）+ Point Light（补光 65W） |
| 镜头 | 焦距 92mm，传感器 32mm |
